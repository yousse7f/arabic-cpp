# توثيق لغة البرمجة العربية (مبنية على C++)

هذه اللغة هي طبقة برمجية فوق لغة C++ تتيح لك كتابة الكود باستخدام مصطلحات عربية مباشرة.

## كيفية البدء

لكي تستخدم اللغة العربية، يجب تضمين ملف `arabic_cpp.h` في بداية ملف الكود الخاص بك:

```cpp
#include "arabic_cpp.h"
```

## قاموس المصطلحات (C++ -> العربية)

### الأنواع الأساسية
| المصطلح في C++ | المصطلح العربي |
| :--- | :--- |
| `int` | `صحيح` |
| `float` | `عشري` |
| `double` | `مزدوج` |
| `char` | `حرف` |
| `void` | `فارغ` |
| `const` | `ثابت` |
| `long` | `طويل` |
| `short` | `قصير` |

### التحكم في التدفق
| المصطلح في C++ | المصطلح العربي |
| :--- | :--- |
| `if` | `إذا` |
| `else` | `وإلا` |
| `for` | `دورة` |
| `while` | `بينما` |
| `do` | `افعل` |
| `break` | `توقف` |
| `return` | `أرجع` |
| `switch` | `تبديل` |
| `case` | `حالة` |
| `default` | `تلقائي` |
| `main` | `الرئيسية` |

### المنطق والمخرجات
| المصطلح في C++ | المصطلح العربي |
| :--- | :--- |
| `true` | `صح` |
| `false` | `خطأ` |
| `cout` | `أخرج` |
| `cin` | `أدخل` |
| `endl` | `إنزل` |

### البرمجة الكائنية والأسماء
| المصطلح في C++ | المصطلح العربي |
| :--- | :--- |
| `class` | `صنف` |
| `struct` | `بنية` |
| `public` | `عام` |
| `private` | `خاص` |
| `protected` | `محمي` |
| `using namespace` | `استخدم_فضاء_الاسماء` |
| `namespace` | `فضاء_الاسماء` |

## مثال كامل

```cpp
#include "arabic_cpp.h"

صحيح الرئيسية() {
    ثابت صحيح العمر = 25;
    
    إذا (العمر >= 18) {
        أخرج << "أنت بالغ" << إنزل;
    } وإلا {
        أخرج << "أنت قاصر" << إنزل;
    }
    
    أرجع 0;
}
```

## ملاحظات هامة
- المجمّع (Compiler) لا يزال C++، لذا يجب الالتزام بقواعد النحو (Syntax) الخاصة بـ C++.
- استخدام الكلمات العربية يسهل فهم المنطق البرمجي للمبتدئين العرب.
