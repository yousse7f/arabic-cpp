# لغة البرمجة العربية (Arabic C++)

لغة برمجة عربية مبنية بالكامل على لغة C++. تهدف هذه المكتية إلى تسهيل تعلم البرمجة للمتحدثين باللغة العربية من خلال استخدام مفردات عربية فصيحة ومباشرة.

## المميزات
- **ترجمة مباشرة**: تحويل الكلمات المفتاحية في C++ إلى العربية (مثل: `صحيح` بدلاً من `int`).
- **سهولة الاستخدام**: يكفي تضمين ملف رأسي واحد (`arabic_cpp.h`) للبدء.
- **كفاءة C++**: البرنامج المكتوب بالعربية يتم تجميعه مباشرة عبر مترجمات C++ القياسية دون فقدان في الأداء.

## كيفية التثبيت
1. قم بتحميل ملف `arabic_cpp.h`.
2. ضعه في مجلد مشروعك.
3. قم بتضمينه في بداية ملف الكود الخاص بك.

## مثال على الكود
```cpp
#include "arabic_cpp.h"

صحيح الرئيسية() {
    أخرج << "مرحباً بك في لغة البرمجة العربية!" << إنزل;
    
    صحيح س = 10;
    صحيح ص = 20;
    
    إذا (س < ص) {
        أخرج << "س أصغر من ص" << إنزل;
    }
    
    أرجع 0;
}
```

## الترجمة المتاحة
| C++ | العربية |
| :--- | :--- |
| `int` | `صحيح` |
| `float` | `عشري` |
| `char` | `حرف` |
| `if` | `إذا` |
| `else` | `وإلا` |
| `for` | `لأجل` |
| `cout` | `أخرج` |
| `endl` | `إنزل` |
| `main` | `الرئيسية` |

## كيفية التشغيل
استخدم مترجم C++ (مثل g++) مع تحديد ترميز UTF-8:
```bash
g++ مثال.cpp -o app.exe -finput-charset=UTF-8 -fexec-charset=UTF-8
./app.exe
```
